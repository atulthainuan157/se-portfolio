# Atul Thainuan — Software Developer Portfolio

A modern, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript. Designed for placement at product-based companies.

## 🔗 Live Demo

> **[View Portfolio →](https://atulthainuan157.github.io/se-portfolio/)**

## ✨ Features

- **Dark Theme** — Professional developer aesthetic with `#000` base and cyan/purple accents
- **Typing Animation** — Auto-cycling role titles (BTech CSE Student, Software Developer, etc.)
- **Glass-morphism Cards** — Subtle backdrop blur, hover glow effects
- **Scroll Reveal** — IntersectionObserver-powered fade-in animations
- **Responsive** — Mobile hamburger menu, stacked layouts on small screens
- **SEO Ready** — Open Graph tags, semantic HTML, meta descriptions

## 📂 Structure

```
se-portfolio/
├── index.html        # Single-page HTML (all 8 sections)
├── style.css         # Dark theme design system
├── script.js         # Typing, scroll-reveal, nav interactions
├── assets/
│   └── resume.pdf    # Resume download
└── README.md
```

## 📑 Sections

| # | Section | Content |
|---|---------|---------|
| 01 | **About** | Professional summary, career goals, interests |
| 02 | **Skills** | C++, Java, Python, JS, React, Node.js, MongoDB, MySQL |
| 03 | **Projects** | 5 project cards with tech stacks & links |
| 04 | **Coding Profiles** | GitHub, LeetCode, Codeforces, GFG, LinkedIn |
| 05 | **Achievements** | Certifications, hackathons, internship, contest rank |
| 06 | **Contact** | Email, phone, LinkedIn, GitHub + contact form |
| — | **Resume** | Download CTA |

## 🛠 Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- Google Fonts — Inter & Fira Code
- Font Awesome 6 icons
- No frameworks, no build step

## 🚀 Deploy on GitHub Pages

```bash
git add .
git commit -m "Initial portfolio"
git push origin main
```

Then in your repo → **Settings → Pages → Source: main branch → Save**

Your portfolio will be live at `https://atulthainuan157.github.io/se-portfolio/`
